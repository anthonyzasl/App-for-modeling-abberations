## 2. Subject and Annotation
**Subject:** Chromatic and spherical aberrations of optical systems

**Annotation:**
The project examines the causes of spherical and chromatic aberrations in optical systems, their calculation methods, and reduction techniques. Using **C#** modeling, the project illustrates spherical lens aberrations and the principles of building achromatic systems.

### Technical Specifications:
* **Programming Language:** C#
* **Development Environment:** Microsoft Visual Studio
* **Compiler Version:** .NET Framework 4.8
