## 2. Тема та анотація
**Тема:** Хроматичні та сферичні аберації оптичних систем 

**Анотація:**
У роботі розглянуто причини виникнення сферичних і хроматичних аберацій оптичних систем, методи їх розрахунку та способи зменшення. За допомогою моделювання мовою **C#** проілюстровано аберації сферичних лінз і принцип побудови ахроматичних систем.

### Технічні характеристики розробки:
* **Мова програмування:** C# 
* **Середовище розробки:** Microsoft Visual Studio 
* **Версія компілятора:** .NET Framework 4.8

## 5. Запуск та встановлення
Програма не потребує складної установки.

### Інструкція із запуску:
1. Скопіювати папку з програмою на комп’ютер.
2. Переконатися в наявності потрібної версії **.NET Framework**.
3. Запустити файл **`WindowsFormsApp2.exe`** подвійним натисканням миші.

> **Особливості введення даних:**
> Для коректної обробки чисел використовуйте **крапку (.)** як роздільник цілої та дробової частин. [cite_start]Використання коми (,) може призвести до помилок.
